GitHub Task Instructions

Step 1: Create and Set Up Your GitHub Account

Go to GitHub and sign up for a new account if you donâ€™t already have one.

Verify your email address and complete the setup process.

Step 2: Create a New Repository

Log in to your GitHub account.

Click the + icon in the top-right corner and select New repository.

Name your repository (e.g., Class_Assignments_FET).

Choose Public (or Private if you prefer).

Optionally, add a README file, .gitignore, and license.

Click Create repository.

Step 3: Collaborate with the Provided Repository

Go to the repository you want to collaborate with: aum-web-app.

Fork the repository to your GitHub account by clicking the Fork button.

Clone the forked repository to your local machine:

git clone https://github.com/your-username/aum-web-app.git cd aum-web-app

Step 4: Pull the Repository to Your Local Machine

If you created a new repository, clone it to your local machine:

git clone https://github.com/your-username/Class_Assignments_FET.git cd Class_Assignments_FET

Step 5: Create a Branch for Task 5 (Calculator Assignment)

Create a new branch for Task 5:

git checkout -b task5_calc_assg

Step 6: Copy Task 5 (Calculator) Files

Copy the Task 5 files into your local repository.

Step 7: Stage and Commit Changes

Stage all changes:

git add .

Commit the changes with a message:

git commit -m "Added Task 5 Calculator Assignment"

Step 8: Push Changes to Remote

Push the changes to the remote repository:

git push origin task5_calc_assg

Step 9: Make a Pull Request

Go to your GitHub repository and create a pull request for the task5_calc_assg branch.

Wait for the pull request to be reviewed and merged.

Step 10: Submit Repository URL and Commit ID

Submit your repository URL and the commit ID for the Task 5 merge in Canvas.

Step 11: Checkout to the Master Branch

Switch back to the main branch:

git checkout main
